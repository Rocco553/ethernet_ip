# EtherNet/IP Documentation

This repository contains documentation and presentations about EtherNet/IP, an industrial fieldbus system.

- **251110_Präsentation_EtherNet_IP.pdf** - EtherNet/IP presentation (PDF format)
- **251110_Präsentation_EtherNet_IP.pptx** - EtherNet/IP presentation (PowerPoint format)
- **Feldbussysteme.pdf** - Fieldbus systems documentation (in German)

## About EtherNet/IP

EtherNet/IP (Ethernet Industrial Protocol) is an industrial network protocol that adapts the Common Industrial Protocol (CIP) to standard Ethernet. It is widely used in industrial automation and control systems.
