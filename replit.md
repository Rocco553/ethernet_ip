# Project: EtherNet/IP Documentation Repository

## Overview
This is a documentation repository containing PDF and PowerPoint files about EtherNet/IP, an industrial fieldbus system. This is not a code project - it's a collection of reference materials and presentations.

## Repository Contents
- Presentation files about EtherNet/IP (PDF and PPTX formats)
- Documentation about fieldbus systems (in German)

## Project Type
- **Type**: Documentation/Reference Materials
- **No Code**: This repository contains no executable code
- **No Workflows Required**: As this is a documentation-only repository, no build or run workflows are necessary
